# 📖 Project Overview

Knowledge Hub is a modern online book delivery management platform that connects readers with librarians through a secure and user-friendly system.

Readers can browse books, request doorstep delivery, pay delivery fees securely using Stripe, and review books after successful delivery.

Librarians can upload and manage their books, while administrators oversee users, books, approvals, deliveries, and platform analytics.

The project is built with modern technologies including Next.js, Express.js, MongoDB, Better Auth, JWT authentication, Stripe Payment Integration, HeroUI, and Tailwind CSS.
